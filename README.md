# Detection-Pipeline-Prototype-

This repository contains a lightweight Python-based detection pipeline and behavioral detection use cases.

## Pipeline
- Ingests and normalizes raw telemetry
- Extracts process, parent, and command-line data
- Applies rule-based detection logic

## Detection Focus
- Encoded PowerShell execution
- Parent-child process anomalies
- Command-line abuse patterns

## Key Challenge
Normalization of inconsistent timestamps and log structures to ensure accurate event correlation.

## Purpose
Demonstrate detection engineering thinking, focusing on behavioral patterns and noise reduction rather than static indicators.
